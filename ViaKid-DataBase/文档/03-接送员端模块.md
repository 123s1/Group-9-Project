## 表结构

### 3.1 qualifications — 五重资质认证表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | UNIQUEIDENTIFIER | PK, DEFAULT NEWID() | 主键 |
| driver_id | UNIQUEIDENTIFIER | NOT NULL, FK(drivers.id) | 关联接送员，一人一type 唯一 |
| qual_type | TINYINT | NOT NULL | 1-身份证认证 2-驾驶证认证 3-行驶证认证 4-无犯罪记录 5-健康证明 |
| qual_name | NVARCHAR(50) | NOT NULL | 资质名称 |
| status | TINYINT | NOT NULL, DEFAULT 0 | 0-待提交 1-审核中 2-审核通过 3-审核驳回 |
| reject_reason | NVARCHAR(255) | | 驳回原因 |
| expire_date | DATE | | 资质到期日 |
| verify_time | DATETIME2 | | 审核通过时间 |
| verify_admin_id | UNIQUEIDENTIFIER | | 审核管理员 ID |
| created_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 创建时间 |
| updated_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 更新时间 |

**索引**：`uk_qualifications_driver_type` (driver_id, qual_type)，`idx_qualifications_driver` (driver_id)

**说明**：与旧文档 `qualification` 表一致，补充了 `verify_admin_id`、`expire_date` 等字段。

---

### 3.2 qualification_documents — 资质材料表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | UNIQUEIDENTIFIER | PK, DEFAULT NEWID() | 主键 |
| qualification_id | UNIQUEIDENTIFIER | NOT NULL, FK(qualifications.id) | 关联资质认证 |
| doc_name | NVARCHAR(100) | NOT NULL | 材料名称（如：身份证正面、身份证背面） |
| doc_url | NVARCHAR(500) | NOT NULL | OSS 文件 URL |
| doc_type | NVARCHAR(20) | NOT NULL | IMAGE / PDF / VIDEO |
| file_size | INT | | 文件大小（字节） |
| sort | INT | NOT NULL, DEFAULT 0 | 排序序号 |
| created_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 创建时间 |

**索引**：`idx_qualification_documents_qual_id` (qualification_id)

---

### 3.3 schedules — 排班表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| driver_id | UNIQUEIDENTIFIER | PK, FK(drivers.id) | 关联接送员 |
| time_slots | NVARCHAR(MAX) | | JSON 格式排班，如 `[{week_day:1, start_time:"08:00", end_time:"18:00", is_available:1, max_order_count:10}]` |
| work_days | NVARCHAR(MAX) | | JSON 工作日数组，如 `[1,2,3,4,5]` 表示周一至周五 |
| unavailable_dates | NVARCHAR(MAX) | | JSON 不可用日期数组，如 `["2026-05-01","2026-05-02"]` |
| max_orders_per_day | INT | NOT NULL, DEFAULT 5 | 当日最大接单数 |
| updated_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 更新时间 |

---

### 3.4 courses — 培训课程表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | NVARCHAR(50) | PK | 课程编号，如 C001 |
| title | NVARCHAR(200) | NOT NULL | 课程名称 |
| description | NVARCHAR(MAX) | | 课程描述 |
| cover_url | NVARCHAR(500) | | 封面图 URL |
| content_type | NVARCHAR(20) | NOT NULL | video-视频 / article-图文 / livestream-直播 |
| content_url | NVARCHAR(500) | | 课程内容 URL |
| duration | NVARCHAR(50) | | 课程时长描述 |
| course_type | NVARCHAR(50) | | safety-安全培训 / service-服务礼仪 / emergency-应急处理 / psychology-儿童心理 / regulation-法规知识 |
| is_required | BIT | NOT NULL, DEFAULT 0 | 是否必修 |
| pass_score | TINYINT | NOT NULL, DEFAULT 60 | 及格分数 |
| sort_order | INT | NOT NULL, DEFAULT 0 | 排序 |
| status | NVARCHAR(20) | NOT NULL, DEFAULT N'active' | active-启用 / disabled-停用 |
| created_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 创建时间 |
| updated_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 更新时间 |

**索引**：`idx_courses_type` (course_type)，`idx_courses_status` (status)

---

### 3.5 course_progress — 培训进度表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| driver_id | UNIQUEIDENTIFIER | PK, FK(drivers.id) | 关联接送员 |
| course_id | NVARCHAR(50) | PK, FK(courses.id) | 关联课程 |
| status | NVARCHAR(20) | NOT NULL, DEFAULT N'not_started' | not_started-未开始 / in_progress-进行中 / completed-已完成 / failed-未通过 |
| progress | TINYINT | NOT NULL, DEFAULT 0 | 学习进度 0-100 |
| last_position | BIGINT | NOT NULL, DEFAULT 0 | 视频播放位置（秒） |
| enrolled_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 报名时间 |
| completed_at | DATETIME2 | | 完成时间 |
| updated_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 更新时间 |

---

### 3.6 exam_questions — 考题表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | NVARCHAR(50) | PK | 题目编号，如 Q001 |
| course_id | NVARCHAR(50) | | 关联课程 ID（可为空表示通用题库） |
| type | NVARCHAR(20) | NOT NULL | single_choice-单选题 / multiple_choice-多选题 / true_false-判断题 |
| content | NVARCHAR(MAX) | NOT NULL | 题目内容 |
| explanation | NVARCHAR(MAX) | | 答案解析 |
| sort_order | INT | NOT NULL, DEFAULT 0 | 排序 |
| created_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 创建时间 |

**索引**：`idx_exam_questions_course` (course_id)

---

### 3.7 exam_options — 考题选项表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | NVARCHAR(50) | PK | 选项编号，如 O001 |
| question_id | NVARCHAR(50) | NOT NULL, FK(exam_questions.id) | 关联题目 |
| option_key | NVARCHAR(10) | NOT NULL | 选项标识 A / B / C / D |
| content | NVARCHAR(MAX) | NOT NULL | 选项内容 |
| is_correct | BIT | NOT NULL, DEFAULT 0 | 是否为正确答案 |
| sort | INT | NOT NULL, DEFAULT 0 | 排序 |

**索引**：`idx_exam_options_question` (question_id)

---

### 3.8 exam_results — 考试成绩表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | UNIQUEIDENTIFIER | PK, DEFAULT NEWID() | 主键 |
| driver_id | UNIQUEIDENTIFIER | NOT NULL, FK(drivers.id) | 关联接送员 |
| course_id | NVARCHAR(50) | NOT NULL | 关联课程 ID |
| total_questions | INT | NOT NULL | 题目总数 |
| correct_count | INT | NOT NULL | 答对题数 |
| score | INT | NOT NULL | 考试成绩（0-100） |
| passed | BIT | NOT NULL, DEFAULT 0 | 是否通过 |
| exam_duration | INT | | 答题用时（秒） |
| answers_json | NVARCHAR(MAX) | | 答题详情 JSON |
| submitted_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 提交时间 |

**索引**：`idx_exam_results_driver` (driver_id)，`idx_exam_results_course` (course_id)

---

### 3.9 fixed_route — 固定线路表

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | UNIQUEIDENTIFIER | PK, DEFAULT NEWID() | 主键 |
| driver_id | UNIQUEIDENTIFIER | NOT NULL, FK(drivers.id) | 关联接送员 |
| route_name | NVARCHAR(100) | NOT NULL | 线路名称（如：朝阳学区线） |
| route_type | TINYINT | NOT NULL, DEFAULT 1 | 1-上学线路 2-放学线路 3-往返线路 |
| start_point | NVARCHAR(255) | NOT NULL | 起点地址 |
| start_lat | DECIMAL(10,6) | NOT NULL | 起点纬度 |
| start_lng | DECIMAL(10,6) | NOT NULL | 起点经度 |
| end_point | NVARCHAR(255) | NOT NULL | 终点地址 |
| end_lat | DECIMAL(10,6) | NOT NULL | 终点纬度 |
| end_lng | DECIMAL(10,6) | NOT NULL | 终点经度 |
| route_points | NVARCHAR(MAX) | | JSON 途经点数组 |
| total_distance | DECIMAL(8,1) | | 总距离（km） |
| estimated_duration | INT | | 预计时长（分钟） |
| week_days | NVARCHAR(20) | | 运营日，如 "1,2,3,4,5" |
| departure_time | NVARCHAR(20) | | 发车时间，如 "07:30:00" |
| status | TINYINT | NOT NULL, DEFAULT 1 | 1-启用 2-停用 |
| created_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 创建时间 |
| updated_at | DATETIME2 | NOT NULL, DEFAULT GETDATE() | 更新时间 |

**索引**：`idx_fixed_route_driver` (driver_id)，`idx_fixed_route_status` (status)

**说明**：阶段一 RouteService 返回模拟数据，表结构已建好，后续接入时直接使用。
